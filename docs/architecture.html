MLOps Platform Architecture

Enterprise ML Lifecycle Management — Model Registry, Drift Detection, Automated Retraining, A/B Testing, Monitoring

Drift Detection Layer

  • - PSI (Population Stability Index) for numerical features
  • - Kolmogorov-Smirnov test for target drift
  • - Chi-square for categorical targets
  • - Threshold: PSI > 0.2 triggers alert
  • - JSON reports saved to disk for audit

Training & Serving

  • - XGBoost models with MLflow tracking
  • - Automated retrain on drift + perf degradation
  • - FastAPI with API key auth + rate limiting
  • - Prometheus /metrics endpoint for monitoring
  • - Model versioning via MLflow aliases

Evaluation & Governance

  • - A/B testing with two-proportion z-test
  • - Cohen's h effect size measurement
  • - Alert manager with severity levels
  • - Grafana dashboard (8 panels, auto-provisioned)
  • - 78 tests across 6 suites, CI/CD on 3 Python versions
